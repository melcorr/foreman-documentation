= Host Management and Monitoring Using {cockpit}

{cockpit} is an interactive web interface that you can use to perform actions and monitor Red Hat Enterprise Linux hosts. You can enable a remote-execution feature to integrate {Project} with {cockpit}. When you install {cockpit} on a host that you manage with {Project}, you can view the {cockpit} dashboards of that host from within the {Project} web UI. You can also use the features that are integrated with {cockpit}, for example, {lorax-composer}.
